# Instrucțiuni Chatbot - Aplicație Curs BNR EUR/RON

## Scop

- adăugare chatbot bazat pe un model LLM în aplicația Streamlit existentă
- chatbotul trebuie să răspundă la întrebări despre cursul valutar EUR/RON
- chatbotul trebuie să poată folosi funcții din aplicație ca tool-uri
- chatbotul trebuie integrat în frontend, într-un widget de chat vizibil în aplicație
- chatbotul trebuie să comunice cu backend-ul FastAPI existent

## Context proiect

Aplicația este deja construită ca proiect full stack pentru prognoza cursului valutar BNR EUR/RON.
